# Task 8 – VPN Usage and Privacy

## 📌 Objective
The objective of this task is to understand how a Virtual Private Network (VPN) enhances online privacy and security by encrypting internet traffic and masking the real IP address.

This task demonstrates VPN usage using Proton VPN and verifies privacy protection through IP address comparison and secure connection checks.

---

## 🛠 Tool Used
- Proton VPN (Free Version)
- Operating System: Windows
- IP Verification Website: https://whatismyipaddress.com

---

## 🔍 Steps Performed

1. Created a free Proton VPN account.
2. Downloaded and installed the Proton VPN client.
3. Checked the public IP address before connecting to the VPN.
4. Connected to a VPN server using Proton VPN.
5. Verified that the IP address and location changed after VPN connection.
6. Confirmed encrypted browsing using HTTPS.
7. Disconnected the VPN and observed IP reverting to original.

## 🔐 Key Observations

- VPN successfully masked the original IP address.
- Internet traffic was encrypted while VPN was active.
- Geographic location appeared different after VPN connection.
- Secure HTTPS connections were maintained.

---

## ⚠️ Limitations of VPN

- Free VPN plans have limited server locations and speed.
- VPN does not provide complete anonymity.
- VPN cannot protect against phishing or malware attacks.

---

## ✅ Conclusion

Using Proton VPN significantly improves online privacy by encrypting traffic and hiding the real IP address. VPNs are essential tools for protecting user data, especially on public or unsecured networks, but should be combined with other security practices for full protection.
